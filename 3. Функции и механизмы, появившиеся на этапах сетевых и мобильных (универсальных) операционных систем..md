# Сетевые ОС
Необходимо иметь возможность дистанционно управлять машиной, узкое место - каналы для передачи
## Передача данных
Используем модемы и витую пару для объединения в сеть
## Появление терминала
Удаленное окно для исполнения команд
## Аутентификация
Большое количество юзеров создало проблемы безопасности - необходимо создавать учетные записи, сеансы, системы авторизации и защиты
# Мобильные системы
Главная проблема - многообразие железа
Решение - Компиляторы, они обеспечивают один универсальный язык для любого железа, которое они поддерживают
