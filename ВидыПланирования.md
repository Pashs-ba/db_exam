# 14. Виды планирования и их место в жизненном цикле процесса.

Горизонт планирования - срок, на который период времени планируется событие
## Виды планирования
1. Краткосрочное планирование (готовность - исполнение)
2. Среднесрочное планирование (готовность вне RAM - готовность, ожидание вне RAM - ожидание прямое)
3. Долгосрочное планирование (рождение - готовность)
4. Планирование очередей ввода-вывода (находится внутри состояния ожидания планирования)
