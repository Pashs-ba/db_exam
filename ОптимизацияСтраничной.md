# Методы оптимизации потребления ресурсов при страничной организации виртуальной памяти. Сегментно-страничная организация виртуальной памяти.

Разбиваем адресное пространство на 6 сегментвов для каждого блока (stack, mmap, heap, noninit init, text). Из каждого сегмента держим соответствующий элемент из pagetable. На это будет узодить меньше памяти (32 мб)
## TLB кеш
Можно задействовать tlb кеш (Буфер ассоциативной трансляции), в котором будем хранить key-value с Nвирт на Nфиз. Он может за 1  обращение найти все ключи (у него всего 64 строки). Нашли - получили значение, иначе - записали.

### Обнуление кеша
Проблема: Охлаждение кеша
Выгодно для длинных процессов

### Вместе с регистрами сохранять кеш
Проблема: Дорого
Выгодно для коротких процессов
