# Цель
ОС существует чтобы обеспечить производительность, надёжность и безопасность, исполнение пользовательских программ, эксплуатирования аппаратного обеспечения, хранения и доступа к данным и диалога с пользователем.
# Функции
1. Управление разработкой и исполнением пользовательского ПО:
	1. API - множество обёрток над системными вызовами
	2. Исполнение.
	3. Обработка ошибок.
	4. Высокоуровневый доступ к вводу-выводу.
	5. Управление хранилищем данных.
	6. Мониторинг.
2. Оптимизация использования ресурсов
3. Поддержка эксплуатации вычислительного узла
4. Поддержка развития
