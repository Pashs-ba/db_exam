# Методы организации хранения данных в файловых системах: индексные дескрипторы.

## EXT2
 Теперь файловая система состоит из:
 1. superblock (хранит инфу о количестве свободных, занятых i-node чтобы сверить количество)
 2. i-node bitmap (свободные ноды, битовый вектор)
 3. block bitmap (свободные блоки, битовый вектор)
 4. i-nodes
 5. data
