Порядок слоёв от центра наружу:
1. Аппаратное обеспечение. 
2. Средства аппаратной поддержки ядра. 
	1. Система прерываний. 
	2. Средство для поддержки привилегированного режима.
	3. Средство поддержки виртуальной памяти.
	4. Смена контекстовых регистров.
	5. Системный таймер.
	6. Защита памяти.
3. Машинно-зависимые модули (hardware abstraction layer, HAL). 
4. Базовые механизмы ядра. Этот слой отвечает за исполнение решений, принятых менеджерами ресурсов. 
5. Менеджеры ресурсов. На этом слое реализованы основные алгоритмы принятия решений: модули для составления расписаний (schedulers) и модули, занимающиеся задачами размещения (allocators).
