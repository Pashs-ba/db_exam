# Центральный сервер
Есть единый memory center, который раздает данные или их адреса всем желающим
Проблемы очевидны

# Миграция Страниц
Воспользуемся идеей свапа. Теперь вместо диска у нас есть другие сервера.
## Плюсы
- Консистентность
- Софт уже почти готов
## Минусы
- Может возникнуть постоянный и долгий обмен страницами между двумя серверами
- Нужно узнавать где находится реальная страница, либо через центральный сервер, либо через "Волну поиска"
# Размножение на чтение
Теперь вместо передачи единственной страницы мы можем делать ее копию.
## Плюсы
 - Скорость
## Минусы
- Память
- Проблемы с записью (нам нужно обновить страницу везде)

# Полная репликация
Решение предыдущей проблемы - если мы хотим записать в страницу то отправляем всем серверам приказ на изменение и данные, которые нужно поменять
Возникнут проблемы с порядком приказов
Решается через центральный сервер, который нумерует запросы
