# Миграция Страниц
Воспользуемся идеей свапа. Теперь вместо диска у нас есть другие сервера.
## Плюсы
- Консистентность
- Софт уже почти готов
## Минусы
- Может возникнуть постоянный и долгий обмен страницами между двумя серверами
