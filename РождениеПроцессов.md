# Методы создания процессов в различных операционных системах. Структуры данных о процессах.

## Принципы
1. Любой процесс порождается процессом. 
2. Process controll block (дескриптор процесса, структура с полями) 
	1.  Идентификация 
	2. Состояние (контекст)
	3. История

В Линкусе при создании процесс либо: 
1. Делает fork(): создаст полную копию с адресным пространством, кодом и тд. Ни один дочерний процесс не будет иметь больше прав, чем его родитель, что выгодно с точки зрения безопасности.
2. Делает exec*(): копирует код родительского скрипта/
