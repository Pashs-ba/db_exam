# Взаимодействие процессов. Условия взаимоисключения и прогресса. Понятие критической секции. Голодание процессов.

## Синхронизация процессов

**Вынужденное взаимодействие** - процессы конкурируют за неразделяемый ресурс. Принтер - неразделяемый ресурс и процессы с прерываниями пытаются записывать в него.

**Внешне-определенное взаимодействие** - обеспечение возможности процессам обменяться данными или управлением.

Синхронизация - попытка достигнуть:
1. **Взаимоисключения.** Неразделяемым ресурсом может в один момент времени пользоваться только один процесс. Наивное решение - lock
2. **Прогресса.** Не должно быть ситуации, когда по всем условиям процесс может выполнится, но не делает этого из-за алгоритма
3. **Отсутствия голодания**. Длительное отсутствие доступа к ресурсу
4. **Отсутствие тупика**. Когда несколько процессов хотят ресурс, который лежит у другого и они не могут дождаться освобождения.


**Критическая секция** - часть кода, которая взаимодействует с неразделяемым ресурсом
